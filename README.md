# 📝 To-Do List App  

##  Features  
-  **Add, edit, delete, and complete tasks**  
-  **Set due dates and priorities** (Urgent, Normal, Low Effort)  
-  **Sort tasks** by:  
  -  **Unfinished status**  
  -  **Due date**  
  -  **Priority**  
-  **Delete completed tasks**  
-  **Data stored persistently with SQLite**
